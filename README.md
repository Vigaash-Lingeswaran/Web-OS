Tamil Cinema OS

This is the Tamil Cinema OS, an OS all about Tamil Cinema, also known as Kollywood. It is themed around the cinema industry and has many details to pay homage to it. Dive in to learn more about this great industry!

Here's the demo link:  https://vigaash-lingeswaran.github.io/Web-OS/
I would recommend viewing the OS with a 75% zoom for the best look!

Features:
- Home Screen: This is the screen everyone is prompted to when they open the OS. It has a brief on the cinema industry and gives a nice welcome message.
- Kollywood Notes: This is a notes app dedicated to Tamil Cinema. Currently, it has notes about Tamil movies and music, and it also gives fantastic recommendations for first-timers.
- Snake Game: This is an app made to keep you relaxed. Play the classic Snake game, collecting as many apples as you can. The controls are just the arrow keys. (The walls don't kill you, only running into yourself does!)

Building Notes:
- The code: The coding languages used for this project were HTML, Javascript, and CSS. HTML was used for the overall OS interface (styles, divs, etc). Javascript was used for the script functions (clicking a button, loading app features. CSS was used for the styles (font size, borders, etc)
- The images: The background was created by me. I compiled a variety of movie posters and created a vintage collage with them. The rest of the images and icons were free-use images.

Running it Locally:
1. Clone this repository
2. Open the cloned folder
3. Open the folder in your editor, and you will have access to all three of the files (index.html, script.js, and styles.css)
4. From there, you can edit whatever things you would like!

I still plan to expand this project by adding more features (more games, a calculator, and better polishing).
